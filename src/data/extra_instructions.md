# Extra Filing Instructions

- Always use the date format **YYYYMMDD** at the beginning of the filename.
- **Avoid using underscores** (`_`) in filenames or directory names unless absolutely necessary (prefer spaces or dashes if specified, but follow the capitalization rule).
- Use **Capitalized Words** (Title case) for filenames and destination folders (e.g., `20240101 New Invoice.pdf`).
